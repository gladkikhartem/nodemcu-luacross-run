# nodemcu-luacross-run
API for NodeMCU LSF cross-compliation  https://nodemcu.readthedocs.io/en/master/lfs/

## Usage
```
curl -d @yourscript.lua -X POST https://nodemcu-luacross-run-64l7ehzjta-uc.a.run.app/compile > output.luac
```
